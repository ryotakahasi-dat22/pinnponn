### ゲームの概要
1. ボールを跳ね返してクリアを目指す対戦ゲーム


### 操作方法
1. マウスを操作
2. ボタンは左クリックで押せる


### 遊び方
1. Timr１０００以内に先に２点をとったほうの勝ちになる。
2. 左にあるバーをボールにあててボールを上手く相手に跳ね返す。
